How to play Pico Fermi Bagel

The game begins with the computer secretly choosing a 3 dgit number with no repeated digits.

You attempt to guess the number.  The computer responds to each guess as follows:
• If the guess has no numbers correct, the response is: "Bagel."
• For each digit the guess has correct, but in the wrong place, the response is: "Pico."
• For each digit the guess has correct and in the correct place, the response is: "Fermi."

As an example, lets say that the secret number is 489.
    Guess 1: 362 Bagel — no digit is correct.
    Guess 2: 820 Pico — the 8 is in the wrong place.
    Guess 3: 418 Pico Fermi — the 8 is in the wrong place, the 4 is in the correct place.
    Guess 4: 518 Pico — the 8 is in the wrong place.
    Guess 5: 487 Fermi Fermi — the 4 and 8 are in the correct place.
    Guess 6: 489 Fermi Fermi Fermi — all digits are in the correct place.

If you cannot correctly guess the number in 20 moves, you lose.  You should be able
to guess the number correctly in 8 moves.
